# SkyRoc 前端菜单分布

> 按生鲜供应链主链路与已实现 API / 权限码整理的推荐侧栏结构。  
> 当前 `DbSeeder` 仍是 Vben 演示菜单（manage / function / document 等），对接前端时按本文写入 `sys_menu` / `sys_menu_button`，再由角色分配后经 `getRoutes` 下发。

## 一级菜单一览

| 顺序 | 一级菜单 | 说明 |
| --- | --- | --- |
| 1 | 首页 | 运营概览 |
| 2 | 销售订单 | 订单列表与详情 |
| 3 | 采购管理 | 计划、采购单、规则 |
| 4 | 库存管理 | 查询、入出库、盘点 |
| 5 | 配送管理 | 任务、异常与基础资料 |
| 6 | 售后服务 | 售后单、取货任务 |
| 7 | 财务管理 | 客户结款、供应商结算 |
| 8 | 商品与定价 | 分类、商品、报价、协议价 |
| 9 | 基础资料 | 客户、供应商、采购员、仓库等 |
| 10 | 质量溯源 | 检测报告、溯源、报送日志 |
| 11 | 数据报表 | 销售/售后/库存/采购汇总 |
| 12 | 系统管理 | 权限、组织、运营与审计 |

## 横切能力（不做一级业务菜单）

| 能力 | 权限 | 建议入口 |
| --- | --- | --- |
| 文件上传 / 下载 | `business:file:read` / `create` | 业务页上传组件内调用 |
| 单据打印 | `business:print:read` / `update` | 单据页内「打印」；模板在系统管理维护 |
| 导入导出 | `business:import-export:read` / `create` | 系统管理，或商品/客户页工具入口 |

## 权限落地说明

- 菜单控制「看见页面」，按钮权限码控制「能不能点」。
- 写操作需对应 `create` / `update` / `delete` / `audit` 等动作权限。
- 非 admin 读报表需配置 `business:report:read` 按钮。
- Admin 持有 `*:*:*` 通配。
- 隐藏详情路由：设置 `HideInMenu` + `ActiveMenu` 指向列表页。

## 主业务链路参考顺序

建档（商品/客户/供应商/仓）→ 报价 / 协议价 → 销售订单审核 → 采购计划 → 采购单 → 采购入库 / 销售出库 → 配送签收 → 客户账单 → 售后 / 退货入库 → 客户结款 / 供应商结算 → 溯源与报表
